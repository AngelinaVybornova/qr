Surplus QR checker
Описание проекта: сканер QR-кода сертификата вакцинации для автоматической проверки его действительности.
Функции:
•	Сканирование QR-кода с подключенной к компьютеру камеры
•	Проверка срока действия сертификата вакцинации при помощи сайта Госуслуг
•	Вывод результата пользователю во всплывающем окне
Проект базируется на задаче №3 компании IT TEST «Автоматизация проверки срока действия QR-кода».
